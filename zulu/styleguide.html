Usage Guide

To incorporate the dark theme into your site:

  • Include the global CSS file in your HTML: 
    <link rel="stylesheet" href="theme.css">
    <link rel="stylesheet" href="elements.css">
  • Use the CSS variables in your styles: 
    color: var(--primary-color);
  • Use predefined classes like .card or .text-muted for consistent styling.
